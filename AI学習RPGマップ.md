# AI学習RPGマップ（AIの世界地図）

AIの世界を「RPGマップ」として見ると理解が楽になります。

## AI WORLD 階層図

```text
AI WORLD
│
├─ Lv1 村（基礎）
│   ├─ AIとは
│   ├─ 機械学習
│   └─ データ
│
├─ Lv2 森（学習方法）
│   ├─ 教師あり学習
│   ├─ 教師なし学習
│   └─ 強化学習
│
├─ Lv3 山（深層学習）
│   ├─ ニューラルネットワーク
│   ├─ CNN
│   └─ RNN
│
├─ Lv4 城（現代AI）
│   ├─ Transformer
│   ├─ LLM
│   └─ 生成AI
│
└─ Lv5 ダンジョン（実践）
    ├─ プロンプト設計
    ├─ RAG
    └─ AIアプリ
```

### ポイント
* AIは巨大に見えるが、構造は意外とシンプル
* **AI** → **機械学習** → **深層学習** → **LLM** という階層を理解するだけで、AI理解の60％はカバーできます。

## 2. Google Skills 攻略ルート（クエスト形式）

Google SkillsをRPGの「クエスト」として進めます。

### クエスト1：AIとは何か
* **理解ポイント**: AIの定義、機械学習との違い、AIの用途
* **アウトプット**: AIとは「コンピューターが人の判断をまねる技術」である。

### クエスト2：機械学習
* **理解ポイント**: データからパターンを学習、モデル、予測
* **アウトプット**: 機械学習とは「大量の例を見てルールを覚える仕組み」である。

### クエスト3：教師あり学習
* **理解ポイント**: 正解付きデータ、分類、回帰
* **例**: 画像 → 猫 or 犬 の判別など。

### クエスト4：ニューラルネットワーク
* **理解ポイント**: 人の脳をまねた構造、入力、重み、出力

### クエスト5：Transformer
* **理解ポイント**: Attention、文脈理解
* **詳細**: ここで ChatGPT, Claude, Gemini などの仕組みが理解できます。

### クエスト6：生成AI
* **理解ポイント**: 文章生成、画像生成、コード生成

## 3. 7日間 AI理解ロードマップ

「全部理解」ではなく、**AIの構造を把握する7日間**のプランです。

| Day      | テーマ       | やること / 理解ポイント                              |
| :------- | :----------- | :--------------------------------------------------- |
| **Day1** | AIとは何か   | Google Skills AI基礎、AIの歴史を学ぶ                 |
| **Day2** | 機械学習     | データ、モデル、学習の概念（データからルールを学ぶ） |
| **Day3** | 教師あり学習 | 分類、回帰（例：スパムメールの判定）                 |
| **Day4** | 深層学習     | ニューラルネットワーク、多層構造                     |
| **Day5** | Transformer  | **Attention、文脈理解（現代AIの核心）**              |
| **Day6** | LLM          | 大量文章で学習した言語モデル（ChatGPT, Gemini等）    |
| **Day7** | AI応用       | RAG、プロンプト、AIアプリの実践                      |

## 4. 学習をポートフォリオ化する方法

GitHubやノートに **AI Learning Log** を作成するのが最もおすすめです。

### 構成案
1. **01 AIとは**
2. **02 機械学習**
3. **03 教師あり学習**
4. **04 深層学習**
5. **05 Transformer**
6. **06 LLM**
7. **07 AI応用**

### 各ページのテンプレート
* **■ 概念**: この技術は何をするものか
* **■ なぜ必要なのか**: どんな問題を解決するのか
* **■ 仕組み**: どのように動くのか
* **■ 具体例**: 実際の使われ方（画像認識、レコメンド等）
* **■ 自分の言葉で説明**: 小学生に説明するつもりで書く
* **■ AIツールとの関係**: ChatGPTやGeminiとの関わり
* **■ 今日の学び / 疑問**: 理解したことと、まだ分からないこと
