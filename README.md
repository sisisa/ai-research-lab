# AI研究所

## プロジェクト概要
AIの最新情報を能動的に収集し、効率的に学習するためのポートフォリオ用Webアプリケーションです。「AI速報ドットコム」などの外部サイトからのスクレイピングと、「Google Skills」との連携を取り入れ、知識のインプットを集約します。さらに、科学的に証明された4つの学習法を取り入れ、高度なUIと洗練されたアニメーションでユーザーの学習体験を劇的に向上させます。

## 技術スタック
- フロントエンド: Next.js (App Router), React, TypeScript
- スタイリング & UI: Tailwind CSS, shadcn/ui, Framer Motion
- バックエンド / データベース: Prisma, SQLite
- スクレイピング: Cheerio (Next.js API Routesにて実装)

## 実装の詳細

### 1. モダンな基本UIと機能紹介（ダッシュボード）
- ダークモードに完全対応した、高いデザイン性を持つダッシュボード。
- 「情報収集」と「効率的学習」の2つの柱をわかりやすく案内する機能紹介UI。
- shadcn/ui の Sidebar コンポーネントを活用したシームレスなナビゲーション。
- Framer Motion を用いたページ全体のフェードインやホバーエフェクト。

### 2. AI情報収集および外部連携 (SearchingHub)
- **AI速報ドットコム連携 (スクレイピング)**: 対象URLを入力することで、Cheerioを用いて実際のWebページからタイトル・要約・本文を自動取得します。
- **データベース保存**: 取得した記事は自動的に SQLite データベースに保存（Upsert）され、履歴として一覧表示されます。
- **インタラクティブな通知**: sonner を採用し、記事取得や学習への追加アクションをリアルタイムでトースト通知します。

### 3. 4つの科学的学習法の実装 (Learning Hub)
- **実データ連携**: SearchingHub で収集した最新のAIニュースを各学習ステップの「問い」として動的に反映。
- **トピック切り替え**: データベースに蓄積された複数の記事を切り替えて、効率的に学習を継続可能。
1. アクティブリコール (Active Recall)
   - 取得した情報を基に自ら記憶を引き出すフラッシュカードUI（Framer Motionによる3D反転アニメーション）。
2. 分散学習・想起学習 (Spaced Repetition)
   - 忘却曲線に基づく復習スケジュール管理。
3. 精緻的質問・自己質問 (Elaborative Interrogation)
   - 記事の内容に対して「なぜ？」を深掘りするテキスト入力フォームとAIフィードバック表現。
4. インターリービング (Interleaving)
   - 異なるジャンルのトピックを混ぜて出題し、柔軟な応用力を鍛えるクイズ形式。

### 4. バックエンドとアーキテクチャ
- **Prisma 7 & LibSQL アダプター**: 
  - 最新の Prisma 7 仕様に適合。
  - LibSQL アダプターを介した接続により、高速かつスケーラブルなデータベース操作を実現。
- **API Routes**:
  - `/api/scrape`: Cheerio を使用した実スクレイピングロジック。
  - `/api/articles`: データベースに保存された記事データの取得。

## セットアップと起動方法

1. 依存関係のインストール
```bash
npm install
```

2. データベースの初期化
```bash
npx prisma generate
npx prisma db push
```

3. 開発サーバーの起動
```bash
npm run dev
```
