# RPG 開発ロードマップ

## 進捗状況: 【■□□□】 25% (Step 1 完了)

このプロジェクトは、2D見下ろし型RPGを通じてAI学習を深めるオープンワールドアプリの開発を目指します。

---

### [x] ステップ1：村の構築と自由移動（基盤作成）
**フロントエンド（Next.js）**
- [x] CSSグリッドを用いた10x10のマップ描画。
- [x] キーボード（WASD/矢印キー）でのキャラクター移動。
- [x] 画面外への移動制限（境界チェック）。

**バックエンド（Python / FastAPI）**
- [x] プレイヤー位置（X, Y座標）の取得・保存API。
- [x] インメモリでの座標データ永続化。
- [x] CORSミドルウェアによるフロントエンド連携の許可。

**完了の目安：**
キャラクターが自由に歩き回り、リロード後もバックエンドから位置を読み込んで再開できる状態。

---

### [ ] ステップ2：住人（NPC）の配置と会話システム
**フロントエンド**
- [ ] マップ上へのNPC配置。
- [ ] 隣接時のアクションキーによる会話ウィンドウUI。

**バックエンド**
- [ ] NPCのプロフィール・セリフデータ提供API。
- [ ] 進行度に応じた動的なセリフ切り替えロジック。

**完了の目安：**
NPCが村に存在し、話しかけると現在の状況に合わせた会話（挨拶など）ができる状態。

---

### [ ] ステップ3：お悩み解決（クエスト）と学習の統合
**フロントエンド**
- [ ] クエスト受注・クリアのフロー実装。
- [ ] 既存のGoogleSkillsコンテンツ（学習画面）とのシームレスな統合。

**バックエンド**
- [ ] クエスト受注状況・クリア条件・スキル習得度のデータ管理（DBテーブルの導入）。

**完了の目安：**
NPCの依頼を解決するために、学習クイズやアドバイス入力をこなし、クリアの報酬（感謝の言葉）を得られる状態。

---

### [ ] ステップ4：世界の変化（オープンワールドの醍醐味）
**フロントエンド**
- [ ] 学習成果に応じたマップの動的変化（新しい建物の出現、道の開通など）。

**バックエンド**
- [ ] 学習履歴に基づいたマップ解放・NPC行動パターンの変化ロジック。

**完了の目安：**
プレイヤーの知識向上に比例して、村が発展し、探索可能な世界が広がっていく状態。
